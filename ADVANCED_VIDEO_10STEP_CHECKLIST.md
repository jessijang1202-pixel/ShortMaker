# 고급영상 10단계 체크리스트 (User-Confirm Mode)

> 이 파일은 **매 단계마다 사용자가 승인**하는 "고급영상" 파이프라인에서 사용합니다.
> AI는 자동으로 판정하고 넘어가지 않습니다. 대신 해당 단계의 체크 항목을 근거로
> "왜 확인이 필요한지"를 짧게 설명하고, 사용자에게 승인/수정 요청/다시 만들기 선택지를 제시한 뒤
> 사용자의 응답을 받고 나서만 다음 단계로 넘어갑니다.

각 단계에서 AI는 다음 형식을 따르세요:
> **확인 요청**: [무엇을 확인해야 하는지]
> **이유**: [체크리스트 근거]
> **선택지**: 승인 / 수정 요청 / 다시 만들기

---

## 1단계 — 주제/컨텍스트 확인
- [ ] 주제가 한 문장으로 명확한가?
- [ ] 핵심 메시지가 15초 이내로 요약되는가?
- **확인 요청 예시**: "핵심 메시지를 'ㅇㅇ'로 잡았는데 이 방향이 맞을까요?"

## 2단계 — 타겟/키 설정
- [ ] 타겟 청중(연령대/관심사)이 지정되었는가?
- [ ] 키(정보성/유머/감성 등) 프리셋이 선택되었는가?
- **확인 요청 예시**: "타겟은 20대 직장인으로 잡고 유머 키로 갈까요, 아니면 다른 방향?"

## 3단계 — 스크립트 이슈
- [ ] 훅이 첫 3초 안에 배치되어 있는가?
- [ ] 러닝타임이 목표 범위 안인가?
- **확인 요청 예시**: "오프닝 한 문장이 두 가지입니다. 어느 쪽이 더 끌리나요?"

## 4단계 — 스크립트 확정
- [ ] 문장당 정보 밀도가 적절한가?
- [ ] CTA가 포함되어 있는가?
- [ ] 저작권·초상권 위험 단어가 없는가?
- **확인 요청 예시**: "이 문장에 브랜드명이 언급되는데 그대로 둘까요, 일반명사로 바꿀까요?"

## 5단계 — 씬 분할 / 스토리보드
- [ ] 씬 시간이 스크립트 흐름과 논리적으로 맞는가?
- **확인 요청 예시**: "6문장을 5개 씬으로 나눴습니다. 이 분할이 자연스러운가요?"

## 6단계 — 비주얼 소스 선정
- [ ] 각 씬의 이미지/영상 소스가 저작권 문제 없는가?
- **확인 요청 예시**: "3번 씬에 쓸 이미지 소스 후보 3개를 골랐어요. 마음에 드는 걸 선택해주세요."

## 7단계 — 텍스트 오버레이 / 자막
- [ ] 씬별 자막 길이가 12자 이내인가?
- [ ] 세로 화면 safe zone 안에 배치되는가?
- **확인 요청 예시**: "이 씬 자막이 조금 길어서 화면에서 잘릴 수 있어요. 줄일까요?"

## 8단계 — 스타일(폰트/컬러) 적용
- [ ] 브랜드 프리셋과 일치하는가?
- **확인 요청 예시**: "폰트를 프리셋 기본값으로 적용했어요. 다른 스타일로 바꾸고 싶으신가요?"

## 9단계 — BGM/효과음 적용
- [ ] 음악 분위기가 콘텐츠 키와 맞는가?
- [ ] 이전 영상들과 키가 급격히 다르지 않은가?
- **확인 요청 예시**: "선택한 BGM이 지난 영상보다 템포가 빠른데, 이대로 진행할까요?"

## 10단계 — 최종 기술 검증 및 렌더링
- [ ] 러닝타임이 목표 범위 ±10% 인가?
- [ ] 자막 타이밍이 음성/영상과 맞는가?
- [ ] 해상도/비트레이트가 플랫폼 기준을 충족하는가?
- **확인 요청 예시**: "최종 길이가 32초로 목표보다 2초 깁니다. 이대로 렌더링할까요, 줄일까요?"

---

### 구현 메모
- 각 단계는 이전 단계 사용자 응답을 반영한 뒤에만 다음 단계 프롬프트를 생성해야 합니다.
- "다시 만들기"를 선택하면 해당 단계만 재생성하고, 이전 단계 결과물은 그대로 유지하세요 (전체 재생성 금지).
- 사용자가 여러 단계를 건너뛰고 싶어 하면("나머지는 알아서 해줘") 남은 단계는 간단영상 모드의 자동 게이트 로직으로 전환해 처리하는 것도 고려해보세요.
